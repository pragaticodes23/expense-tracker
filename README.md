# Expense Calculator (Python)

A beginner Python project that calculates total expenses and the remaining balance after spending in different categories.

---

## 🚀 Features
- Takes a total budget amount as input
- Records spending on:
  - Food
  - Travel
  - Shopping
  - Entertainment
- Calculates total amount spent
- Calculates remaining balance
---

## 🧠 Concepts Used
- Variables
- User Input
- Strings
- Arithmetic Operators
- Basic Calculations
- Output Formatting
---
## 💻 Sample Run

### Input

Enter total amount: 10000

Enter expenditure on food: 2000

Enter expenditure on travelling: 1500

Enter expenditure on shopping: 3000

Enter expenditure on Entertainment: 500

### Output

Total amount spent is: 7000

Total balance amount left is: 3000

Thank you for using EXPENSE CALCULATOR

---

## 🎯 Purpose

This project was built to practice Python fundamentals, user input handling, variable usage, and arithmetic calculations through a real-world budgeting scenario.
